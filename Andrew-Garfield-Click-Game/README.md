# Andrew Garfield Click Game

A simple browser-based mini game developed as part of my studies in the Entra21 program.

The project was created during my first steps with HTML, CSS and JavaScript, focusing on DOM manipulation, click events, timers, animations and basic game logic.

## About the Project

In this game, the player needs to click on the Andrew Garfield images when they appear above the boxes. Each successful click increases the score.

The game uses random timers to make each target appear at different moments, creating a simple reaction-based challenge.

## Technologies Used

- HTML
- CSS
- JavaScript

## Features

- Interactive click-based gameplay
- Score system
- Random target appearance
- Simple animation using CSS transitions
- Visual click effect
- DOM manipulation with JavaScript

## Project Status

This is a beginner study project.

The main goal was not to create a perfect or fully responsive game, but to practice the fundamentals of front-end development and build something interactive using JavaScript.

## How to Run

1. Download or clone the repository.
2. Open the `index.html` file in your browser.
3. Click on the characters when they appear.
4. Try to get the highest score possible.
